Предварительные АТД:
1) Поле:
Матрица прямоугольной формы с размером по Х и У, может заполняться разными юнитами или объектами, быть разных размеров,
 обладать разными качествами (например иметь бортики или нет)
2) Юнит:
Имеет размер по Х и по У, прямо
3) Действие - ввод клавиши пользователем
4) Событие - столкновение юнитов, попадание объекта в юнит, генерация юнита итд. Все события на поле
5) Игра - отслеживает все юниты, объекты, поле, евенты, действия, по сути - "мозг"
