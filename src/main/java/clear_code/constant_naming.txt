1. CHILD_MULTIPLIER - вынес константа множителя для поиска дочернего элемента в aBST
2. LEFT_CHILD_SUMMAND - вынес слагаемое для поиска левого дочернего элемента в aBST
3. RIGHT_CHILD_SUMMAND - вынес слагаемое для поиска правого дочернего элемента в aBST

6. PARAM1/2/3/4 -> *_FIELD_ID - переименовал неочевидные константы в более понятный вид
7. WRONG_STATUSES - вынес некорректные статусы заявки (отказ скоринга etc)
8. SECOND_HASH_MULTIPLIER - вынес мультипликатор второго хеша в переменную в фильтере блума
9. INIT_ARR_SIZE - вынес дефолтный начальный размер массива в ArrayList;
10. NOT_FOUND_EXCEPTION - вынес текст ошибки в константу
11. AUTH_TOKEN_HEADER_NAME - вынес название токена авторизации в переменную
12. PATH - вынес дефолтный путь к контроллеру
