game:
  Описание: АТД, управляющая игровым процессом, включая инициализацию игры, основной цикл, ввод пользователя и определение конца игры
  Функции: Запуск и завершение игры, обработка ввода пользователя, обновление игрового состояния, подсчет очков и ходов

board:
  Описание: АСД, представляющая собой игровое поле размером 8x8, заполненное элементами
  Функции: Инициализация и отображение игрового поля, проверка и выполнение перестановок элементов, обработка совпадений, обновление состояния поля после удаления элементов

element:
  Описание: АСД, представляющая элемент на игровом поле
  Функции: Хранение типа элемента, предоставление типа элемента

position:
  Описание: АСД, представляющая координаты элемента на игровом поле
  Функции: Хранение координат элемента, преобразование пользовательского ввода в координаты на поле

score:
  Описание: Сущность, управляющая подсчетом очков в игре
  Функции: Подсчет очков за удаленные элементы, предоставление текущего количества очков


score strategy:
  Описание: АСД, определяющая стратегию подсчета очков за совпадения на игровом поле
  Функции: Расчет очков в зависимости от количества совпадений

element factory (Фабрика элементов):
  Описание: АСД, создающая элементы для игрового поля
  Функции: Генерация новых случайных элементов для заполнения игрового поля
