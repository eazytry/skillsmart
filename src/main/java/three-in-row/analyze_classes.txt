Классы анализа:

Game: Управляет игровым процессом, включая инициализацию игры, основной игровой цикл, ввод пользователя и определение конца игры

Board: Представляет собой сетку, заполненную элементами, и содержит логику проверки и выполнения перестановок, а также обработки совпадений элементов

Element: Представляет собой элемент на игровом поле, который может быть перемещен и участвовать в совпадениях

Position: Хранит координаты элемента на игровом поле, создается из пользовательского ввода

Score: Управляет подсчетом очков в игре, хранит и обновляет текущее количество очков

Move: Представляет собой действие по перестановке двух соседних элементов на игровом поле, проверяет допустимость хода

GameOverChecker: Проверяет наличие возможных ходов и определяет, когда игра должна завершиться

InputHandler: Обрабатывает ввод пользователя

Statistics: Ведение статистики игры, включая подсчет очков, количество ходов и вывод итоговой статистики в конце игры

Bonus: Представляет собой бонус, который может быть активирован при выполнении определенных условий на игровом поле

BonusEffect: Определяет эффекты, которые вызываются активацией бонуса (например, удаление строки/столбца/элемента, взрыв области 2х2)
