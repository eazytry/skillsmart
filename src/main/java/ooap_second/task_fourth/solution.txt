Базовый класс Window:

Описание: Абстрактный класс/интерфейс, который определяет интерфейс для всех графических окон
Почему закрыт: Этот класс закрыт для изменений, чтобы обеспечить стабильный интерфейс для всех подклассов.
Изменения в этом классе могут повлиять на все производные классы.

Описание: Конкретные реализации класса Window. Каждый из этих классов реализует методы, определенные в Window.
Например: PopupWindow, FullScreenWindow, WidgetWindow etc..
Почему открыты: Эти классы открыты для расширения, что позволяет добавлять новые формы без изменения существующего кода.
Можно легко добавить новый класс, например, FullScreenPopupWindow, следуя тому же шаблону.

Класс WindowRenderer:
Описание: Класс, отвечающий за отрисовку графических окон. Он может принимать объекты типа Window и вызывать их методы
отрисовки.
Почему закрыт: WindowRenderer закрыт для изменений, так как его основная функциональность - работать с любыми объектами
типа Window. Изменение этого класса потребует изменений во всех классах, использующих его.

Принцип Открыт-Закрыт в этой иерархии позволяет добавлять новые типы фигур без необходимости изменять существующие
классы.
Это обеспечивает гибкость в расширении функциональности и уменьшает риск внесения ошибок при модификации существующего
кода.
