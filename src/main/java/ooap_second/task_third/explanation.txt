В разработке на ЯП Java, использование готового кода возможно с помощью следующих инструментов:
- Библиотеки как зависимости, которые сборщик проекта (gradle, maven) скачивает и добавляет в classpath при
  упаковке проекта в jar
- При разработке core ф-сти разработчики Java использовали API, которое предоставляет сам ЯП Java, для внедрения
  ф-ций/модулей, написанных на C++/других более низкоуровневых (необязательно) языках. Например метод System.arrayCopy(..).
  Такие ф-ции помечаются ключевым словом "native".
