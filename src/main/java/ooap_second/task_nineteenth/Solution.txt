1) subtypes inheritance:
- Запчасть -> Кузовная запчасть/Запчасть трансмиссии/Запчасть двигателя
- Бутылка -> Стеклянная бутылка/Пластиковая бутылка

2) restriction inheritance:
- Врач -> Хирург/Анастезиолог/Акушер (врач-анастезиолог не может проводить операции)
- Программист -> Java/Python/C# программист (накладываются ограничения на язык, на котором будет писать программист)
- Помещение -> Коммерческое помещение/Жилое помещение

3) extension inheritance:
- Список -> Однонаправленный список -> Двунаправленный список
- Телефон -> Сенсорный телефон
- Процессорное охлаждение -> Водяное процессорное охлаждение
